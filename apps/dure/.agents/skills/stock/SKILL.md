# Stock — 종목 시세 및 수급

## 실행 방법

    cd apps/dure && npm run start -- call <method> '<json_params>'

---

## 현재가 / 기본정보

### `stock.current_price` — 현재가 + 투자지표

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| `stk_cd` | string | O | 종목코드 (6자리, e.g. "005930") |

**반환 주요 필드**: 현재가, 전일 대비, 등락률, PER, PBR, 시가총액, 기관 순매수

## 수급 / 투자자 동향

### `stock.investor` — 투자자별 매매 동향

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| `stk_cd` | string | O | 종목코드 |
| `date` | string | | 조회 기준일 (YYYYMMDD, 기본: 최근 거래일) |

**반환**: 기관, 외국인, 개인, 연기금, 보험 등 투자자 유형별 순매수/순매도

**해석 가이드**:
- 외국인 순매수 + 기관 순매수 동반: 강한 매수세 (긍정적)
- 외국인 순매도 + 개인 매수: 스마트머니 이탈 (주의)
- 연기금 순매수: 저가 매수 또는 장기 보유 신호

## 사용 예시

```bash
# 삼성전자 현재가 조회
npm run start -- call stock.current_price '{"stk_cd":"005930"}'

# SK하이닉스 호가 조회
npm run start -- call stock.order_book '{"stk_cd":"000660"}'

# 기관/외국인 매매 동향
npm run start -- call stock.investor '{"stk_cd":"005930"}'

# 애널리스트 투자의견
npm run start -- call stock.investment_opinion '{"stk_cd":"005930"}'
```
