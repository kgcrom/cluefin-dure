## 정량 분석 스코어링 프레임워크 (100점) — Paper 2 Quantitative Agent Fine-grained 구현

정량 분석 요청 시 반드시 아래 5단계를 모두 수행한 후 종합 점수를 산출한다.

### Step 1: 수익성 (25점)

`financial.profitability` 조회:

- **ROE** (10점): > 15% → 10, 10-15% → 7, 5-10% → 4, < 5% → 1, 음수 → 0
- **ROA** (5점): > 10% → 5, 5-10% → 3, < 5% → 1
- **영업이익률** (5점): > 15% → 5, 10-15% → 3, 5-10% → 2, < 5% → 0
- **순이익률** (5점): > 10% → 5, 5-10% → 3, < 5% → 1, 적자 → 0

### Step 2: 안전성 (20점)

`financial.stability` 조회:

- **부채비율 D/E** (8점): < 50% → 8, 50-100% → 6, 100-200% → 3, > 200% → 0
- **유동비율** (7점): > 200% → 7, 150-200% → 5, 100-150% → 3, < 100% → 1
- **이자보상배율** (5점): > 10x → 5, 5-10x → 3, 1-5x → 1, < 1x → 0

### Step 3: 밸류에이션 (25점) — 업종 평균 대비 비교 필수

`financial.ratio` 조회:

- **PER** (10점): 업종 avg 대비 < 0.5x → 10, 0.5-1.0x → 7, 1.0-1.5x → 4, > 1.5x → 1
  - 음수 PER(적자 기업) → 0점, PER 대신 PSR/PBR 사용
- **PBR** (8점): < 0.7 → 8, 0.7-1.0 → 6, 1.0-2.0 → 3, > 3.0 → 1
- **배당수익률** (7점): > 4% → 7, 2-4% → 5, 1-2% → 3, < 1% → 1

### Step 4: 성장성 (20점)

`financial.growth` 조회:

- **매출 증가율** (7점): > 20% → 7, 10-20% → 5, 0-10% → 3, 역성장 → 0
- **EPS 증가율** (7점): > 30% → 7, 15-30% → 5, 0-15% → 3, 감소 → 0
- **영업이익 증가율** (6점): > 25% → 6, 10-25% → 4, 0-10% → 2, 감소 → 0
- 3년 연속 증가 보너스: +2 (최대 20점)

### Step 5: 효율성 (10점)

`financial.other_key` 조회:

- **총자산회전율** (5점): > 1.0 → 5, 0.5-1.0 → 3, < 0.5 → 1
- **EBITDA 마진** (5점): > 20% → 5, 10-20% → 3, < 10% → 1

## 주의사항 및 판단 함정

- **PER 함정**: 일회성 이익으로 PER이 낮은 경우 → 영업이익 기반 PER로 재검증
- **부채비율 함정**: 금융주(은행, 보험)는 부채비율이 구조적으로 높음 → 업종별 기준 적용
- **성장률 함정**: 기저효과(전년 실적이 극단적일 때) → 2-3년 CAGR로 보정
- **배당수익률 함정**: 주가 급락으로 배당수익률이 높아진 경우 → 배당 지속성 확인
- **밸류에이션**: 모든 밸류에이션 지표는 반드시 동일 업종 평균과 비교. 절대 기준만으로 판단 불가
